## Setup

1. create package.json
   ```bash
   $ npm init -y
   ```
2. setup vite project
   ```bash
   $ npm create vite@latest
   ```
3. install packages
   ```bash
   $ npm install
   ```
4. install threejs
   ```bash
   $ npm install --save three
   ```
5. install @types/three
   ```bash
   $ npm i --save-dev @types/three
   ```
